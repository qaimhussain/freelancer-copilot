# Freelancer Copilot

A Chrome extension that scans Upwork job posts and returns AI-powered 
bid recommendations, proposal drafts, price ranges, and red flag warnings.

Built in one night as part of Comeback Pakistan Cohort 1.

## Setup
1. Clone this repo
2. Add your own API keys to config.js (see config.example.js)
3. Go to chrome://extensions → enable Developer Mode → Load unpacked → select this folder
